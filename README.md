# Helios

Helios is a small language model project built from scratch as a learning journey into AI and machine learning.

The goal of the project is to understand how modern language models work by building and training one step by step rather than simply using existing APIs.

## Current progress:

* Custom tokenizer trained on a book corpus
* Custom language model training pipeline
* Multiple training runs completed
* Trained on classic literature datasets
* Wikipedia integration in progress

## Current model size:

* ~1.8 million parameters

## Current training status:

* 20+ training epochs completed

## Project goals:

* Improve language quality
* Expand knowledge using Wikipedia
* Improve conversational abilities
* Continue experimenting with model architectures

This repository documents the development of Helios from its earliest versions.
